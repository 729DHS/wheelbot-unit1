# 2026-05-04 — Spring-Damper 分支初始化

## 分支
`feature/spring-damper-sync`

## 变更内容

### feat: spring-damper control with side detection (80bfe6d)
- 替换自适应增益 (KP ramp 30→200) 为固定低力弹簧阻尼 (KP=10, KD=0.5)
- 控制律: `torque = KP*(home - pos) + KD*(0 - vel)`，可手动转动，松手回弹
- 新增侧边检测: 左腿(电机1,2) vs 右腿(电机3,4)
- 修复 `hold_kp`/`hold_kd` 从标量改为 per-motor 数组

### fix: enable CONFIG_CBPRINTF_FP_SUPPORT (7985a6c)
- 开启浮点 printf 支持，修复串口输出 `*float*` 问题

### fix: hysteresis + debounce for side detection (2f6f85f)
- 侧边检测改为双阈值滞回: ON=0.08 rad, OFF=0.03 rad
- 新增 80ms 去抖动过滤机械耦合误触发
- 阈值可通过 OpenOCD 实时调整 (`cmd_side_on_thresh`, `cmd_side_off_thresh`)

### docs: polar coordinate findings (d5a864e)
- 新建 `08-弹簧阻尼极坐标发现.md`: 角度 φ 回弹好，距离 r 回弹差
- 原因: 五连杆耦合非线性映射，电机角度回弹 ≠ 末端极坐标回弹
- 更新 INDEX.md, 07-TODO.md
