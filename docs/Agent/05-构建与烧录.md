# 构建与烧录

## 环境

- Zephyr 3.7.0: `/home/huiming/zephyrproject/zephyr`
- Zephyr SDK 0.17.0: `/opt/zephyr-sdk-0.17.0`
- 工具链: `arm-zephyr-eabi-gcc` 12.2.0
- 目标板: `robomaster_c` (DJI RoboMaster Type-C, STM32F407IG)

## 构建

```bash
cd /home/huiming/桌面/Projects/Unit1

export ZEPHYR_BASE=/home/huiming/zephyrproject/zephyr
export ZEPHYR_SDK_INSTALL_DIR=/opt/zephyr-sdk-0.17.0

# 配置
cmake -B build -DBOARD=robomaster_c -DBOARD_ROOT=.

# 编译
cmake --build build
```

## 烧录 (CMSIS-DAP / OpenOCD)

```bash
openocd -f interface/cmsis-dap.cfg -f target/stm32f4x.cfg     -c "adapter speed 1000"     -c "program build/zephyr/zephyr.elf verify reset exit"
```

## 调试 (OpenOCD + GDB)

```bash
# 终端1: 启动 OpenOCD 服务
openocd -f interface/cmsis-dap.cfg -f target/stm32f4x.cfg

# 终端2: GDB 连接
arm-zephyr-eabi-gdb build/zephyr/zephyr.elf     -ex "target extended-remote localhost:3333"     -ex "monitor reset halt"
```

### 修改变量示例

```
(gdb) set var cmd_kp = 200       # 增大刚度
(gdb) set var cmd_kd = 3         # 增大阻尼
(gdb) p g_dm4310.online_mask     # 查看在线状态
(gdb) p g_dm4310.motor[0].pos_rad # 查看电机1位置
```
