# 弹簧阻尼测试：极坐标回弹的发现

## 时间
2026-05-04

## 测试结果

### 极坐标定义
将五连杆末端（轮子中心）相对动力轴用极坐标 (r, φ) 描述：
- **φ (角度)**: 回弹效果良好，松手后能稳定回到原位
- **r (距离)**: 回弹效果差，松手后回不到原位甚至偏差很大

### 原因分析
DM4310 电机在 MIT 模式下做的是**各电机独立的角度弹簧阻尼**：
```
torque_i = KP * (home_angle_i - actual_angle_i) + KD * (0 - vel_i)
```

但五连杆机构中，电机角度与末端极坐标 (r, φ) 是**耦合非线性映射**：
- 单独某个电机转动 → 同时影响 r 和 φ
- 两个电机配合转动 → 可以实现纯 r 或纯 φ 变化

这意味着对电机角度做弹簧阻尼 ≠ 对末端极坐标做弹簧阻尼。
