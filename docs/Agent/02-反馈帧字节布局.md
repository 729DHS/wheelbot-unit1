# 反馈帧字节布局

## 说明书 vs 实际协议

《DM-J4310-2EC V1.1 说明书》描述的反馈帧格式与实际通信抓包存在字节偏移差异。

### 说明书描述 (第1节)

```
D0=ID | D1=ERR | D2-D3=POS | D4-D5=VEL | D5-D6=T | D7=reserved
```

### 实际协议 (C++ 参考实现 & 通信抓包)

```
D0[7:4]=state | D0[3:0]=ID | D1-D2=POS | D3-D4=VEL | D4-D5=T | D6=T_MOS | D7=T_Rotor
```

## 差异对照表

| 字段 | 说明书偏移 | 实际偏移 |
|------|-----------|---------|
| 电机 ID | D0 全字节 | D0 低4位 |
| 电机状态 | D1 | D0 高4位 |
| 位置 POS | D2-D3 | D1-D2 |
| 速度 VEL | D4-D5 | D3-D4 |
| 扭矩 T | D5-D6 | D4-D5 |
| MOS 温度 | 独立帧 | D6 |
| 线圈温度 | 独立帧 | D7 |

## 正确解码代码

```c
motor_id = (uint8_t)(data[0] & 0x0FU);
out->motor_state = (uint8_t)(data[0] >> 4);
p_int = ((int)data[1] << 8) | (int)data[2];      // D1-D2 = POS
v_int = ((int)data[3] << 4) | ((int)data[4] >> 4); // D3-D4 = VEL
t_int = (((int)data[4] & 0x0F) << 8) | (int)data[5]; // D4-D5 = T
out->mos_temp = data[6];   // MOS温度
out->coil_temp = data[7];  // 线圈温度
```

## 结论

以 C++ 参考实现和实际通信抓包为准, 说明书可能描述的是不同固件版本。
